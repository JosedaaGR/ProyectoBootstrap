Itinerario formativo CFGS "DAW"

A continuación os mostramos las orientaciones para elegir el itinerario formativo que el Departamento de Informática del IES Ruiz Gijón recomienda seguir al alumnado que cursa el Ciclo "Desarrollo de Aplicaciones Web" en oferta parcial, tal y como queda recogido en la Programación Educativa del Ciclo:


MÓDULOS PROFESIONALES CON FORMACIÓN BÁSICA O DE SOPORTE Y SU RELACIÓN CON LOS DE 2º CURSO (por tanto, es necesario haber superado el de primero):

  • Sistemas Informáticos (1º DAW) -> Despliegue de Aplicaciones Web (2º DAW)
  • Bases de Datos (1º DAW) -> Desarrollo Web en Entorno Servidor (2º DAW)
  • Programación (1º DAW) -> Desarrollo Web en Entorno Servidor y Desarrollo Web en Entorno Cliente (2º DAW)
  • Lenguajes de Marcas y Sistemas de Gestión de Información (1º DAW) -> Desarrollo Web en Entorno Cliente, Desarrollo Web en Entorno Servidor y Diseño de Interfaces Web (2º DAW).


MÓDULOS PROFESIONALES CON FORMACIÓN COMPLEMENTARIA ENTRE AMBOS (se aconsejan cursar simultáneamente):

  • Programación (1º DAW) + Entornos de Desarrollo (1º DAW)
  • Desarrollo Web en Entorno Cliente (2º DAW) + Desarrollo Web en Entorno Servidor (2º DAW) + Despliegue de Aplicaciones Web (2º DAW) + Diseño de Interfaces Web (2º DAW)


MÓDULOS PROFESIONALES CON FORMACIÓN TRANSVERSAL:

  • Lenguajes de Marcas y Sistemas de Gestión de Información
  • Sistemas Informáticos
  • Bases de Datos
  • Programación
  • Entornos de Desarrollo
  • Formación y orientación laboral
  • Empresa e iniciativa emprendedora
